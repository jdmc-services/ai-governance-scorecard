# AI Governance Readiness Assessment

**A free, interactive AI governance assessment for regulated industries — scored against the frameworks that matter in your environment.**

Live demo → [ai-governance-scorecard.streamlit.app](https://ai-governance-scorecard.streamlit.app)

---

## Overview

This tool helps organizations understand their AI governance posture before deploying AI in regulated environments. Visitors select their industry, answer 15 questions across 5 governance domains, receive a scored risk tier, and get a personalized gap analysis generated by Claude.

Built and maintained by [JDMC Services](https://jdmcservices.com) — AI Governance for Regulated Industries.

## How It Works

1. **Select your industry** — Healthcare, Education, Financial Services, or Energy
2. **Answer 15 questions** — Yes, Partial, or No across 5 governance domains
3. **Get your score** — 0 to 30 across four risk tiers
4. **Receive your gap analysis** — Claude generates a personalized 3-paragraph report with prioritized next steps specific to your industry and scores

### Scoring Tiers

| Score | Tier | Label |
|---|---|---|
| 0–6 | 1 | Foundational |
| 7–10 | 2 | Developing |
| 11–14 | 3 | Managed |
| 15–30 | 4 | Optimized |

## Local Development

```bash
git clone https://github.com/jdmc-services/ai-governance-scorecard.git
cd ai-governance-scorecard
pip install -r requirements.txt
```

Create a `.streamlit/secrets.toml` file:

```toml
ANTHROPIC_API_KEY = "sk-ant-your-key-here"
```

Run the app:

```bash
streamlit run app.py
```
